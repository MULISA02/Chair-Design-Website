# Chair-Design-Website
A modern, responsive landing page for a boutique furniture store. This project was built to demonstrate proficiency in semantic HTML structure, custom CSS design systems, and responsive web design.

## Features
* **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens using CSS Grid and Flexbox.
* **Semantic Structure:** Built with accessibility and SEO in mind using clean, modern HTML5.
* **Design System:** Implemented a consistent visual language including custom typography and a curated color scheme.
Interactive UI: Includes hover effects and a structured product gallery.
Tech Stack
* **HTML5:** Structure and content.
* **CSS3:** Custom styling, layouts, and animations.

## What I Learned
How to build a complex layout from scratch without relying on heavy frameworks.
The importance of a "mobile-first" approach in modern web development.
Managing layout alignment and spacing through professional CSS techniques.
